# AULA - GIT

### COMANDOS BÁSICOS

Autenticar via computador local no GitHub
```
gh auth login
```

Clonar repositório no GitHub: 
```
gh repo clone nome-do-repositório
```
Abri o VS Code:
```
code . 
```

### Salvar no GitHub

Preparar arquivosd para salvar: 
```
git add .
```

Comando para versionar (salvar):
```
git commit -m "mensagem"
```

Baixar atualizações do GitHub:
```

git pull
```
Enviar para GitHub:
```
git push
```
